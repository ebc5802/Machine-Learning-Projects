# HW 2 — Support Vector Machines

Written assignment covering the theory and mathematics of Support Vector Machines.

## Topics Covered
- **Hard-margin SVM** — formulating the maximum-margin classifier as a constrained optimization problem
- **Soft-margin SVM** — introducing slack variables to handle non-separable data
- **Dual formulation** — Lagrangian duality and the Karush-Kuhn-Tucker (KKT) conditions
- **Kernel trick** — mapping data to higher-dimensional feature spaces for non-linear classification (RBF, polynomial kernels)
